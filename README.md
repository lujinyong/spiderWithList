基于大神的代码之上做封装
* 增加快手某用户的作品列表爬取
* 增加网页文件解析爬取（因未实现分页爬取，可以手动加载完所有视频后，统一爬取）

主程序：extract.py   
运行后输入参数分类两种：
* http://开头格式直接自动爬取网页
* C:/dir/**.html(windows)|~/Desktop/files/xxx.html(mac|linux) 直接解析文件

* #### 都是相对简单的爬虫，熟练应该看一眼就懂了，如果是初学者，里面有些东西还是值得看一看的。

* ####  爬虫文件详情在这里 [extractor](/extractor)
---
```shell
pip3 install -r requirements.txt
python3 extract.py
```
可能还需要安装nodejs
